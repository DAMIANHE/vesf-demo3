# VESF DEMO — Versión Institucional

Este repositorio contiene el DEMO institucional del framework
**VESF — Value Early Signals Framework**.

## Objetivo del DEMO
- Mostrar el marco conceptual de detección temprana de riesgo invisible
- Educar a Directores y Alta Gerencia
- Preparar la transición hacia la versión comercial
- Proteger la propiedad intelectual (sin fórmulas ni motor real)

## Qué SÍ hace este DEMO
- Visualiza riesgos estratégicos en estado temprano
- Simula escenarios (Conservador / Base / Tensión)
- Muestra impacto VAN simbólico
- Exporta PDF en modo preview
- Captura leads de forma invisible

## Qué NO hace este DEMO
- No calcula VAN real
- No expone fórmulas
- No muestra ponderaciones internas
- No guarda datos

## Estructura clave
- `/app/page.tsx` → Home institucional (incluye imagen conceptual)
- `/app/demo` → Flujo completo del DEMO
- `/components` → Componentes visuales
- `/public/vesf-home-visual.png` → Imagen HOME (obligatoria)

## Instrucciones de despliegue (Vercel)
1. Borrar proyecto anterior
2. Subir este repositorio completo
3. Verificar que exista `/public/vesf-home-visual.png`
4. Deploy sin modificar código

No agregar cálculos reales ni lógica adicional en el DEMO.
La versión comercial es un proyecto separado.
